Seleccionar partes do texto

Escreva ou copie o texto para a caixa esquerda na página Textos. Selecione cada trecho e prima enter para o usar no diagrama.

Desenhar o diagrama

Posicione cada trecho arrastando com o rato. Para posicionar todo o diagrama pode arrastar o fundo.

Um duplo-click selecciona um trecho. Um click com o botão da direita desliga esse trecho do diagrama.

Com um trecho seleccionado, aponte o rato para outro trecho e prima:

  • R para o designar como razão para o trecho seleccionado.
  • + para criar uma conjunção com o trecho seleccionado.

Com um trecho seleccionado prima P para o mudar entre razão e pressuposto (premissa implícita).

Apontando o rato para um trecho prima O para abrir ou fechar a arvore do diagrama nesse ponto ou V para alterar a validade da inferência que se segue.

Outras teclas: B para alterar a cor do fundo; Shift+X para apagar o diagrama (mantendo o texto).

Selecting from a text

Type or paste text into the Text box in the Texts page. Select each statement to use in the diagram and press enter.

Drawing the diagram

Place each statement by dragging with the mouse. Drag the background to scroll the diagram.

Double-Click to select a statement. Right-click to disconnect a statement.

With one statement selected, point the mouse to another statement and press:

  • R to set the statement as a reason.
  • + to create a conjunction.
  • P to set the statement as a pressuposition (implicit).
  • O to open or close the tree from the statement pointed to.
  • V to toggle the validity of the inference.
