       Tool for representing arguments in a diagram
	                  Version: 1.0
	   
  This software is meant to help explain argument in a 
  classroom. Copy-paste the text, select the elements of 
  the argument, and drag them to represent their logical
  relations.
  The main motivation was my dislike for writing on the
  board, especially with chalk.
  
  Instructions in the Help.html file
